# 🍔 Food Landing Page

Landing Page desenvolvida utilizando **HTML5** e **CSS3**, com foco em um design moderno, responsivo e intuitivo para apresentação de um restaurante ou serviço de delivery.

## 🚀 Tecnologias

* HTML5
* CSS3

## 🎯 Objetivo

Este projeto foi desenvolvido para praticar a criação de interfaces modernas utilizando HTML e CSS, aplicando conceitos de responsividade, organização de layout e boas práticas de desenvolvimento front-end.
